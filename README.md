# 快速上手

> 构建图片标记


## CDN 引入 inmark
``` bash

<script type="text/javascript" src="http://www.wulihub.com.cn/go/JPVkkW/inmap_grey.min.js"></script>
```


## NPM 安装
推荐使用 npm 来安装，享受生态圈和工具带来的便利，更好地和 webpack 配合使用，当然，我们也推荐使用 ES2015。

``` bash
$ npm install inmark --save
```
